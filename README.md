# VeriTubes

VeriTubes es una plataforma web de videos verificados, donde los usuarios pueden:

- Iniciar sesión con Google
- Votar si un video es Verdadero, Falso o Parcial
- Ver porcentajes en tiempo real de los votos
- Consultar información adicional sobre cada video

## Tecnologías utilizadas

- React + Vite
- Firebase Authentication (Google)
- Firebase Firestore (para votos y videos)
- Vercel para hosting y despliegue
